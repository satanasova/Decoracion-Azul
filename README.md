# Decoracion-Azul

### Description
  Blog with one internal blog page

### Built With
  * Sass
  * Bootstrap
  * JS 
  
### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/satanasova/Decoracion-Azul.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the project
   ```sh
   npm start
   ```
